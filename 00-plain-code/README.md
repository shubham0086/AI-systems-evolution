# Rung 00 : Plain Code

**No model. No intelligence. Just instructions a human wrote.**

This is the baseline. The program does exactly what it was told and nothing more.
Change the topic and the output stays generic, because there is no understanding
behind it : only hard-coded text.

- **Adds:** determinism, speed, zero cost.
- **Can't:** handle anything it wasn't explicitly programmed for.

```bash
node main.js
```

**You use this every day without thinking about it:** a pocket calculator, a spell-checker that matches fixed rules, a search bar that matches exact keywords. No model behind any of it. Just code doing exactly what a human told it.
